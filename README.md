VoiceAnalyst (Аудио-Скрайбер)

Телеграм-бот, который расшифровывает аудио и видео, выделяет главные мысли и создаёт готовый протокол.

Что умеет бот?

Отправь боту голосовое, аудио или видео — и получи:

- **Текст расшифровки** — Whisper переведёт речь в текст
- **Темы, решения, задачи** — YandexGPT выделит главное
- **Mind Map** — карта связей между темами
- **Протокол .docx** — готовый документ для рассылки
- **Статистику речи** — сколько слов, какой темп
- **Тайм-коды** — в какой момент какая тема обсуждалась
- **Перевод** — на 7 языков
- **Аудиодорожку** — можно скачать отдельно
- **Обрезку видео** — указал время, получил анализ фрагмента

---

Как запустить у себя:

1. Скачай проект:
git clone https://github.com/Hitguber228/Textonator.git
cd Textonator

2. Установи библиотеки:
pip install -r requirements.txt

3. Создай файл `config.py` и запиши туда ключи:
python
TELEGRAM_BOT_TOKEN = "токен от BotFather"
YANDEX_API_KEY = "ключ от Yandex Cloud"
YANDEX_FOLDER_ID = "ID каталога"
FFMPEG_PATH = r"C:\ffmpeg\bin\ffmpeg.exe"
FFPROBE_PATH = r"C:\ffmpeg\bin\ffprobe.exe"

4. Установи FFmpeg с ffmpeg.org (Без него никак) Вот гайд: https://www.youtube.com/watch?v=9_ldCQUgU7Q

5. Запусти!
python main.py

---

Как пользоваться?
1. Найди бота в Telegram: @VoiceAnalystBot
2. Отправь файл
3. Выбери "Анализировать полностью" или "Обрезать фрагмент"
4. Получи результат

---

Стек технологий:
Python 3.11
Faster-Whisper (распознавание речи)
YandexGPT (анализ текста)
NetworkX + Matplotlib (Mind Map)
python-docx (протоколы)
FFmpeg (обработка медиа)
python-telegram-bot (интерфейс)
